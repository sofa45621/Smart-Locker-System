# 📦 Smart Face Locker System (智慧人臉辨識置物櫃)

這是一套基於 Python 與 OpenCV 開發的「商業機台級」智慧置物櫃系統。
本專案透過高精度人臉辨識技術取代傳統實體鑰匙或密碼，並結合 STM32 單晶片進行硬體繼電器控制，打造出兼具安全性、隱私保護與防呆機制的無接觸式 Kiosk 終端機台。

## ✨ 核心特色功能 (Key Features)

* **👀 活體偵測防護 (Liveness Detection)**
  導入 68 點臉部特徵點分析，計算眼睛長寬比 (EAR)。使用者必須對著鏡頭「眨眼」以證明為真人，徹底防堵照片或手機螢幕破解攻擊。
* **🧠 最佳匹配演算法 (Best Match Algorithm)**
  捨棄傳統的「先到先得」比對法，系統會一次性計算畫面人臉與資料庫所有佔用者的「128維特徵距離分數」，精準找出相似度最高的使用者，完美解決長相相似導致的誤判問題。
* **🛡️ 閱後即焚隱私保護 (Privacy First)**
  當使用者成功取物後，系統會觸發清除機制，將該使用者的臉部特徵資料與照片從本機硬碟及記憶體中「徹底銷毀」，確保個資不外洩。
* **🛑 防資源濫用機制 (Anti-Hogging)**
  在存物註冊前進行反向查核，若系統發現該使用者「已經佔用了一個櫃位」，將會無情拒絕其重複註冊請求，強制要求先取物再存物。
* **⏱️ Kiosk 閒置超時防呆 (Timeout Reset)**
  當機台進入確認畫面超過 10 秒無人操作時，系統將自動終止當前流程並退回主畫面，確保機台永遠處於可服務下一位客人的狀態。
* **🔐 管理員超級後臺 (Admin Override)**
  內建隱藏的後門指令與密碼驗證系統。遭遇突發狀況時，管理員可強制發送指令開啟指定櫃位，並清空該櫃位資料。

## 🛠️ 系統架構與硬體需求 (Architecture & Hardware)

* **前端辨識核心**: Python 3, OpenCV, `face_recognition` (基於 dlib), Numpy
* **硬體控制單元**: STM32 單晶片 (透過 USB 序列埠接收 Python 發送的 `[Oxx]` 字串指令，驅動繼電器開門)
* **影像擷取設備**: 網路攝影機 (WebCam) 或透過 DroidCam 串流之手機鏡頭
