# 📦 AI 臉部辨識智慧置物櫃 (Smart Kiosk Locker)

本專案是一個結合邊緣運算 (Edge Computing) 與微控制器 (MCU) 的 AIoT 智慧置物櫃系統。透過 Python 進行即時的人臉特徵提取，並搭配「狀態機」邏輯，實現類似車站、百貨公司 Kiosk 機台的動態存取物功能。系統透過 UART 與 STM32 開發板連動，精準控制實體電子鎖。

## 🌟 系統核心特色 (Key Features)

* **互動式 Kiosk 介面 (UI/UX)**：具備「主選單、存物確認、取物確認」三段式狀態機，並提供 `Y/N` 雙重防呆確認，避免使用者誤觸。
* **動態多人分配 (Multi-User Support)**：拋棄傳統固定名單，支援隨機客即時註冊。系統可同時管理多個佔用中的置物櫃，自動分配專屬櫃號。
* **絕對隱私保護 (Burn-After-Reading)**：採用「閱後即焚」機制。使用者成功取物後，系統會瞬間銷毀硬碟中的實體照片與記憶體中的臉部特徵，確保個資零殘留。
* **斷電記憶恢復 (Auto-Recovery)**：系統啟動時會自動掃描本機端未取物的紀錄，即使遭遇停電或程式重啟，客人的存物權限也不會遺失。
* **軟硬體非同步控制**：Python 負責高算力的人臉比對，辨識成功後透過 UART (115200 bps) 發送訊號，交由 STM32 負責大電流繼電器與電控鎖的實體驅動。

## 🛠️ 硬體架構 (Hardware Requirements)

* **微控制器**：STM32 Nucleo 開發板
* **影像來源**：智慧型手機 (透過 DroidCam 串流) 或 USB 網路攝影機
* **控制元件**：5V 繼電器模組 (Relay)
* **執行機構**：DC 12V 伸縮電控鎖 (Solenoid Lock)

## 💻 軟體與依賴套件 (Software Dependencies)

* **開發環境**：Python 3.x
* **核心套件**：
  * `opencv-python` (即時影像擷取與 UI 繪製)
  * `face_recognition` (人臉定位與 128 維特徵矩陣比對)
  * `pyserial` (與 STM32 進行跨平台硬體通訊)
  * `shutil` (負責系統檔案的動態銷毀)

## 🔄 系統操作流程 (Operation Flow)

1. **[ 空閒狀態 ]**：機台顯示主選單，等待客人操作。
2. **[ 存物 Store ]**：客人按下 `s` 進入存物模式，正對鏡頭並按下 `y` 確認。系統自動拍攝照片、分配櫃號、解鎖空櫃，並進入佔用狀態。
3. **[ 取物 Open ]**：客人按下 `o` 進入取物模式，正對鏡頭並按下 `y` 確認。系統在動態資料庫中尋找相符的特徵，若比對成功即解鎖對應櫃門，並徹底刪除該客人的所有資料。
